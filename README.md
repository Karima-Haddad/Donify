# 🩸 Donify

Donify is a blood donation management platform that connects donors, hospitals, and AI-powered prediction services to reduce blood shortages and optimize emergency response.

---

## 🚀 Vision

To build a smart, scalable, and data-driven ecosystem that improves blood donation coordination using real-time matching and AI prediction.

---

## 🏗 System Architecture

Donify follows a modular microservice-ready architecture:

Frontend (React + Vite + TypeScript)
⬇
Backend API (Node.js + Express)
⬇
Database (PostgreSQL - Supabase Cloud)
⬇
AI Service (Python - Planned)

---

## 📂 Project Structure

project-root/
│
├── backend/ → Node.js API server
├── frontend/ → React application
├── ai/ → AI prediction service (planned)
├── docs/ → Architecture & documentation
└── README.md

## ⚙️ Technologies

- Frontend: React + Vite + TypeScript
- Backend: Node.js + Express
- Database: PostgreSQL (Supabase Cloud)
- AI: Python (FastAPI - planned)
- Version Control: Git + GitHub

## 🚀 Getting Started

### 1️⃣ Clone the repository

git clone <repo-url>
cd project-root


---

### 2️⃣ Backend Setup

cd backend
npm install
npm run dev


Server runs on:
http://localhost:4000

---

### 3️⃣ Frontend Setup

cd frontend
npm install
npm run dev


Frontend runs on:
http://localhost:5173
