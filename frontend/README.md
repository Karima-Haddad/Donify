# 🎨 Donify Frontend

React + Vite + TypeScript application for Donify.

---

## 🏗 Stack

- React 18
- Vite
- TypeScript
- Axios for API calls

---

## 📂 Structure



frontend/
│
├── src/
│ ├── components/
│ ├── pages/
│ ├── services/
│ ├── App.tsx
│ └── main.tsx
│
├── .env
└── package.json


---

## 🔧 Frontend Installation

Create project:

  npm create vite@latest .

Install dependencies:

  npm install



## 🚀 Backend Setup

  cd backend  
  npm install  

---

## ▶️ Run Development Server

npm run dev

App runs at:
http://localhost:5173


---

## 🧠 Frontend Guidelines

Keep components reusable

Separate pages and components

Use TypeScript interfaces

Centralize API calls

Avoid hardcoded URLs
